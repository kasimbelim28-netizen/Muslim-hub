# Muslim Hub

A Discord login website with dashboard and Supabase database.

## Features
- Discord OAuth login
- User dashboard
- Admin panel (owner only)
- Partnership request page
- Supabase database storage

## Pages
- index.html → login page
- dashboard.html → user dashboard
- admin.html → admin panel
- partnership.html → partnership form

## Setup
1. Add your Discord CLIENT_ID in auth.js
2. Add your Supabase URL + anon key in supabase.js
3. Deploy on Netlify
